TumblrAccount Form

Steps to generate tumblr API keys

Register with Tumblr : https://www.tumblr.com/register

Create an application: https://www.tumblr.com/oauth/register

Use the following details to register an application.

Application Name: - anything of your choice

> Application website: https://www.google.com

> Application description: “this application to broadcast into different social media from one application”

Default Callback url: http://localhost:5000

Application id: anything your choice

Click on register

Now, click show secret key.

Note down both OAuth consumer key and secret key.

Got to the link https://api.tumblr.com/console/calls/user/info and provide your consumer key and secret and click on authenticate.

Click on allow access for the application name you entered.

Click on show keys at the top right corner and make a note os all the keys.

Please enter these api keys in the above form to continue

This is a one time process and after submitting the apI keys, you will be able to make posts to Tumblr anytime after this.
